About This Project

This project aims to predict the artist of a given painting using a deep learning model. The model has been trained on a dataset of paintings from various artists.

Steps to Use the Application

  1. Go to the Home page.
  2. Upload an image of a painting using the provided upload button.
  3. Wait for the model to process the image and predict the artist.
  4. View the predicted artist and the prediction probability displayed on the screen.
